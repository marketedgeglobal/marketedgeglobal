How to Update Ranking Data

Quick guidance for non-technical editors.

Home / Explore Platform / Ranking Tool / How to Update

Where to edit data

Update metrics by editing JSON at /data/ranking-tool/*.json.

criteria.json scores.json value_chains.json

Field descriptions

Field Description
id Short unique key used across files (no spaces).
label Human-readable name shown on the dashboard.
area Criteria Area label used for detail filters.
weight Percent weight for a criterion (0-100).
score Raw score for a value chain + criterion (numeric).
weighted_score Optional override for the computed score.
explanation Short rationale shown in the detail view.

How to add a Criterion Area or Value Chain

Add a new entry to criteria.json or value_chains.json, then add matching rows to scores.json.

New criterion IDs must be unique. New value chains need scores for each criterion.

Preview changes

Use the GitHub web editor to update JSON, commit the change, and refresh the page.
